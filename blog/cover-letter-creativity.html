창의성, 어떻게 자소서에 쓸 수 있을까

"저는 창의적인 사람입니다"는 아무런 설득력이 없습니다. 창의성은 주장이 아니라 행동으로 보여줘야 합니다. 남들이 생각하지 못한 방법으로 문제를 해결한 경험, 기존 방식에 의문을 품고 새로운 접근을 시도한 경험이 창의성의 증거입니다.

창의성 표현의 2가지 유형

유형 1 — 새로운 방법으로 문제 해결

"학과 홍보 담당을 맡았는데 포스터를 아무도 보지 않는다는 것을 발견했습니다. 타겟이 모이는 카페와 식당에 QR코드를 붙이고, 참가 신청을 오픈카톡으로 받는 방식으로 바꿨습니다. 홍보 예산은 그대로였지만 행사 참여자가 전년 대비 2.3배 늘었습니다. 홍보는 만드는 것이 아니라 닿게 하는 것이라는 것을 배웠습니다."

유형 2 — 기존 방식에 의문을 품고 개선

"편의점 아르바이트에서 재고 관리가 수기로 이루어지고 있었습니다. 매번 오류가 생기고 시간이 많이 걸렸습니다. 엑셀로 간단한 입출고 관리 시트를 만들었고, 재고 파악 시간이 30분에서 5분으로 줄었습니다. 점장님이 다른 지점에도 적용하겠다고 하셨습니다."

직무별 창의성·문제해결 예시 5개

마케팅

"SNS 광고 성과가 낮았던 이유를 분석했더니 타겟팅이 아닌 콘텐츠 형식의 문제였습니다. 정적 이미지를 15초 쇼츠로 바꾸었더니 클릭률이 3배 올랐습니다."

기획

"회의가 매번 결론 없이 끝나는 문제를 발견했습니다. 회의 전 '오늘 결정해야 할 것 3가지'를 공유하는 규칙을 제안했고, 평균 회의 시간이 90분에서 40분으로 줄었습니다."

개발

"반복되는 배포 과정을 수동으로 처리하는 것이 비효율적이라고 판단했습니다. GitHub Actions로 자동화 파이프라인을 구축했고, 배포 시간을 40분에서 5분으로 단축했습니다."

운영

"창고 정리를 맡았을 때 물건을 찾는 데 매번 10분 이상 걸리는 문제가 있었습니다. 사용 빈도 기준으로 물건을 재배치하고 위치 라벨을 붙였더니 평균 탐색 시간이 2분으로 줄었습니다."

CS·서비스

"같은 문의가 반복되는 것을 발견하고 FAQ 문서를 직접 만들었습니다. 팀원들과 공유했더니 답변 시간이 절반으로 줄었고, 고객 만족도도 높아졌습니다."
